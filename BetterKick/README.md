# Better Kick

This mod makes it so you can help your friends **and** kick them at the same time!
The kick now uses a keybind (Default is `V`).

This mod is **All Clients** meaning that everyone must have the mod installed to kick.

## Trolling

Everyone needs this mod to be kicked, meaning troll however you want!
IF you can kick, everyone has the mod installed, so they want to kick too!
